Demo 4 Elemen HTML Keren

sebenernya ini lagi bahas samp, tapi yaudahlah sekalian, bahas pre, code, samp sama kdb

"Elemen <samp> (Sample Output) dalam HTML digunakan untuk menampilkan output dari program, sistem, atau perangkat lunak.
Teks di dalamnya biasanya ditampilkan dalam font monospace (seperti <code> dan <kbd>), sehingga mudah dibedakan dari teks biasa."

Contoh Penggunaan yang Benar & Cantik

Ketik perintah di terminal, lalu tekan Enter:

$ git status

Maka output yang akan muncul:

On branch main
Your branch is up to date with 'origin/main'.
nothing to commit, working tree clean

Contoh output error satu baris:

Error 404: Halaman tidak ditemukan.

Contoh dalam kalimat biasa:

Program mengembalikan: Hello, World! Selamat ya!

Contoh kombinasi lengkap (paling sering dipakai di tutorial pro):

$ python halo.py
Halo, selamat datang di dunia Python!
Umur kamu berapa? 17
Wah masih remaja nih!

Tips:

  • Gunakan <pre><code> → untuk perintah yang diketik user
  • Gunakan <pre><samp> → untuk output multi-baris
  • Gunakan kbd → untuk tombol keyboard
  • Jangan ubah font-family di body atau :root kalau mau dan tetap kelihatan beda
